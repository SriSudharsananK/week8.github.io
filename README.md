# Quarterly Earnings Report - Reveal.js Presentation

This is an interactive presentation created using Reveal.js for the Q4 2025 Quarterly Earnings Report.

## Features

*   **Email:** Includes the presenter's email address.
*   **Markdown Slides:** Demonstrates the use of Markdown for creating slide content.
*   **Animated Elements:** Uses fragments for animated bullet points.
*   **Code Syntax Highlighting:** Shows a Python code sample with syntax highlighting.
*   **Mathematical Equations:** Displays the Black-Scholes formula using KaTeX.
*   **Speaker Notes:** Includes speaker notes for presentation guidance (press 's' to view).

## How to View

1.  Open the `index.html` file in your web browser.
2.  Navigate through the presentation using the arrow keys.
3.  Press the 's' key to open the speaker notes view.

## How to Publish on GitHub Pages

1.  **Create a new GitHub repository.**
2.  **Push the project files (`index.html` and this `README.md`) to the repository.**
3.  **In your repository's settings, go to the "Pages" section.**
4.  **Under "Branch", select the branch you pushed your code to (e.g., `main` or `master`) and click "Save".**
5.  **Your presentation will be live at `https://<your-username>.github.io/<your-repository-name>/`.** 
    *   **Note:** It may take a few minutes for the site to be deployed.
