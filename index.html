Code Example: Calculating Loan Payments


def calculate_monthly_payment(principal, annual_rate, years):
    """Calculates the monthly payment for a loan."""
    monthly_rate = annual_rate / 12 / 100
    num_payments = years * 12
    
    payment = principal * (monthly_rate * (1 + monthly_rate)**num_payments) / ((1 + monthly_rate)**num_payments - 1)
    
    return payment

# Example usage:
loan_amount = 500000
interest_rate = 5.0
loan_term = 30

print(f"Monthly Payment: ${calculate_monthly_payment(loan_amount, interest_rate, loan_term):.2f}")

Financial Modeling: Black-Scholes

The Black-Scholes formula is used to model the price of European-style options.

\[ C(S, t) = N(d_1)S - N(d_2)Ke^{-r(T-t)} \]

Where \(d_1 = \frac{\ln(\frac{S}{K}) + (r + \frac{\sigma^2}{2})(T-t)}{\sigma\sqrt{T-t}}\) and \(d_2 = d_1 - \sigma\sqrt{T-t}\)

This is the famous Black-Scholes model. We use it extensively in our derivatives pricing. It looks complex, but it's a cornerstone of modern financial theory.
